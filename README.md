# Limit.less 🔗

**Limit.less** is a high-end, minimalist web application designed for secure, ephemeral text sharing. It allows users to share sensitive information—passwords, API keys, or private messages—that automatically self-destructs after a specified number of views.

Built for privacy-conscious individuals and teams who need a "no-history" way to transmit data.

## ✨ Features

- **Self-Destruction:** Links automatically expire after a set number of views (1-6).
- **Ephemeral Storage:** Data is stored in-memory; once it's gone, it's purged from the server forever.
- **Copy-to-Clipboard:** Seamless interaction with one-click copy for links and payloads.
- **High-End UI:** A stark, minimalist monochrome aesthetic with professional typography and analogue-grain textures.
- **Privacy-First:** No tracking, no databases, no persistent logs of your secrets.

## 🚀 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Modern CSS-in-JS (Global CSS with variables)
- **State Management:** React Hooks & In-Memory Store
- **Language:** JavaScript (ESM)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/emmsdan/limit-less.git
   cd limit-less
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
